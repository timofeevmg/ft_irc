# ft_irc
irc server on C++ for Textual client

School 21 group project with jmarian(https://github.com/pokeska1) / rmerrie(https://github.com/RuslanBairashev)

Реализованные вызовы:
* PASS
* NICK
* USER
* TOPIC
* VERSION
* INFO
* NOTICE
* KICK
* PRIVMSG
* JOIN
* ISON
* MODE
* INVITE
* PART
* WHO
* PING
* file transfer support
